
# 统计与指标（Summary）
突出展示指定维格列的统计值和目标值，作为关键指标

## 🎨 功能介绍

统计与指标小程序（下文简称 「统计与指标」 ）可以统计表格内任一列数据，并将统计值以突出的样式显示在小程序上。你可以用来展示一些关键指标的统计值和目标值，比如销售额、订单数、注册用户数、BUG数……

详细介绍见官方帮助手册：[「统计与指标」小程序 | Vikadata](https://vika.cn/help/intro-widget-summary/)
